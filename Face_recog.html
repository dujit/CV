DESCRIPTION

At First, We have imported all the necessary modules of Face_Recognition, Os, and OpenCV in our .py script. When the user submits his mailid, the System will check if the user id is Present in the database. If it's not , then the System will generate the output as "You Have not Registered yet". But if it's present in the database, the system will store the image in a variable and the face_locations function will detect the face using HOG(Histogram of Oriented Gradients ) Algorithm. Using the face_encodings Function 128 Different Features of the face will be stored in a variable. and after that, the System will compare the Encoded Face with the user's face. If the faces match, the system will generate the output as "Authentication Successful" else system will generate the output as "Unauthorized Access".
